Column names and types in a dataframe.

Usage

types_df(df)

Arguments

df
A dataframe.

Value

A tibble with the column names and types.

Description

Given a dataframe, types_df returns a tibble with the column names and types.

Examples

(mtcars %>%
   types_df())
