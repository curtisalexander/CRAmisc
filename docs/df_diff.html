Test how dataframes are the same / different.

Usage

df_diff(x, y)

Arguments

x
First dataframe.
y
Second dataframe.

Value

Test results and differences are printed to standard out.

Description

df_diff prints out how dataframes are the same and how they are different.

Details

Tests include

  • Equal # of rows
  • Equal # of columns
  • Row names are the same
  • Column names are the same
  • Class(es) is / are the same
  • Column types are the same - work in progress
  • Values within each column are the same

df_diff is unstable at the moment and should be considered a work in progress.

Examples

x <- tibble::tribble(
 ~col1, ~col2,
 1, "a",
 2, "b",
 3, "c"
)

y <- data.frame(col1 = 1:3, col2 = c("a", "b", "d"), col3 = c(10, 11, 12))

df_diff(x, y)
