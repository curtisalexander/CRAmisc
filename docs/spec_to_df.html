Convert spec to a dataframe

Usage

spec_to_df(spec)

Arguments

spec
A spec to be written as a dataframe

Value

A tibble representation of a spec.

Description

Take a spec and converts it to a dataframe. The dataframe can then be written out as a CSV in lieu of writing out an actual rds object. This is useful for cases when reading a spec that was saved as an rds throws an error.

Details

After reading the specification in CSV format back into R, it can be converted into an actual specification using col_spec_update. See the example for methodology.
