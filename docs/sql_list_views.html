List views.

Usage

sql_list_views(db_con, db_name, db_schema)

Arguments

db_con
Database server connection object.
db_name
Database name.
db_schema
Database schema.

Value

A character vector of all views.

Description

sql_list_views lists all views in a Microsoft SQL Server database.

Details

Will not return tables. If tables and views are desired, use sql_list_tvs.
